# nimrtl

[ref: #module-nimrtl]

Main file to generate a DLL from the standard library. The default Nimrtl does not only contain the system module, but these too:

* parseutils
* strutils
* parseopt
* parsecfg
* strtabs
* times
* os
* osproc
* unicode
* pegs
* ropes
* cstrutils
